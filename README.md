## Description <a id="description"></a>

A user enters three numbers into the console application and the application determines if the those numbers were lengths of a three sided geometric shape if that shape would be a triangle. If the shape is a triangle the application determines which kind of triangle. The application prints the results to the console.

## Setup/Installation Requirements <a id="setup"></a>

* Open the terminal on your local machine
* If [C#](https://docs.microsoft.com/en-us/dotnet/csharp/) and [.NET](https://docs.microsoft.com/en-us/dotnet/) are not installed on your local device, follow the instructions [here](https://www.learnhowtoprogram.com/c-and-net-part-time-c-and-react-track/getting-started-with-c/installing-c-and-net)
* Navigate to the directory inside of which you wish to house this project
* Clone this project with the following git command `$ git clone <https://github.com/jessicarubinwilliams/TriangleTracker.Solution>`
* Navigate to the top level of the repository with the command `$ cd TriangleTracker.Solution`
* Navigate to the test subdirectory with the command `$ cd TriangleTracker.Tests`
* Recreate project environment and install required dependencies with terminal command `$ dotnet restore`
* Run tests in MSTest with the command `$ dotnet test`
* Navigate to the production project directory with the command `$ ../TriangleTracker`
* Run the program in the console with the command `$ dotnet run`
